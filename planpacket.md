ada dua plan di dalam projek ini 
1. plan untuk akun user (Free,starter, pro, enterprice)
2. plan untuk workspace (Free, starter, pro, enterprice)

1. untuk plan akun user 
    - free : mendapatkan 1 workspace.
    - starter : mendapatkan 3 workspace
    - pro : mendapatkan 10  
    - enterprice : mendapatkan 20 workspace
    - custom : mendapatkan sesuai permintaan

2. untuk plan workspace 
    - free : 3 content type 500 entris, 100MB Storage, 1 user, 
    - starter : 5 content type 5000 entris, 1GB Storage, 3 user
    - pro : 10 content type 10000 entris, 5GB Storage, 10 user
    - enterprice : 20 content type 20000 entris, 10GB Storage, 20 user 
    - custom : mendapatkan sesuai permintaan


3. premium oddons 
    - extra storage 
    - extra content type 
    - extra user 

4. AI Feature 
    - content generator
    - ai frontend generator 


5. openclaw untuk generate frontend dengan boot python yang bisa kita gunakan untuk memilih model ai.
